## ⚡ Quick Start (3 Steps)

### Step 1: Install Dependencies
```bash
cd FRONTEND
npm install
```
⏱️ Takes 2-3 minutes

### Step 2: Start Server
```bash
npm run dev
```

### Step 3: Open Browser
Visit: **http://localhost:3000**

✅ **You should see the Cerevia landing page!**

## 💡 Key Points to Remember

### 1. Why Next.js?
"File-based routing and server-side rendering for better performance"

### 2. Why Tailwind?
"Utility-first CSS that's faster than writing custom stylesheets"

### 3. Why Components?
"Each component has one job, making code reusable and maintainable"

### 4. Flexbox vs Grid?
"Flexbox for 1D layouts like navbars, Grid for 2D layouts like cards"

### 5. Where's the Data?
"Using sample data now, backend team building API with Prisma + PostgreSQL"

## 🐛 Troubleshooting

### "Port 3000 in use"
```bash
npm run dev -- -p 3001
```

### "Module not found"
```bash
rmdir /s node_modules
npm install
```

### "Tailwind not working"
```bash
# Restart server (Ctrl+C then)
npm run dev
```
